    📂 变更文件 — 文件状态说明
    
  
    点击文件名可查看具体变更内容（diff）
    
  
    
    ??未跟踪（新文件）
    M已修改
    A已暂存的新文件
    D已删除
    R重命名
    U合并冲突
  
    

    



    
  
    📜 提交记录 — 回滚模式说明
    
  
    点击 commit hash 或回滚按钮可选择回滚模式
    
  
    
    soft
    仅移动 HEAD 指针，代码和暂存区不变
    mixed
    移动 HEAD + 清空暂存区，代码保留
    hard
    ⚠ 彻底还原，未提交的修改全部丢失
