# DESIGN.md — Git Save/Load

参考自 Linear 设计系统（awesome-design-md）。
适配为 Git 存档/读档工具的 UI 规范。

## 色板

- 底色: `#010102`（近黑带微蓝）
- 表面阶梯: `#0f1011` / `#141516` / `#18191a`
- 主色: `#5e6ad2`（lavender-blue）
- 文字: `#f7f8f8` / `#d0d6e0` / `#8a8f98`
- 边框: `#23252a`
- 成功绿: `#27a644`

## 设计原则

- 深色主题，不提供浅色模式
- 主色仅用于：主按钮、焦点环、品牌标识、链接强调
- 层次用表面阶梯 + 边框线表示，不用阴影
- 卡片圆角 12px，按钮 8px，输入框 8px
- 正文用 14px/16px，等宽字体 13px
- 间距基准 4px，卡片内边距 16-24px

## 核心布局

插件聚焦两个核心动作：**存档（Save）** 和 **读档（Load）**

```
┌─────────────────────────────┐
│  Git Save / Load            │
│  ┌────────┐ ┌────────────┐  │
│  │ SAVE   │ │  LOAD       │  │  ← 两个主按钮
│  │ 一键存档│ │ 提交历史     │  │
│  └────────┘ └────────────┘  │
│                             │
│  当前分支: master            │
│  变更: 3 个文件              │
│  ┌──────────────────────┐   │
│  │ 存档消息: [        ]  │   │  ← 存档输入
│  │ [一键存档] [暂存]     │   │
│  └──────────────────────┘   │
│                             │
│  ─── 最近提交 ───            │
│  a1b2c3d feat: xxx          │
│  e4f5g6h fix: yyy           │
└─────────────────────────────┘
```
